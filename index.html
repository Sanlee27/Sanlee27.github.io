해당 날짜의 접속자(counter) 테이블에 추가되어 일일/누적 접속자 수에 합산되도록 하였습니다.
가계 내역 작성 시, 수입과 지출 내역을 구분 선택하고
메모에 해시태그(#)를 붙혀 작성하면, 메인 상단에 해당 키워드가 표시되도록 했습니다.

달력
- JAVA Calendar API를 활용, 달력을 활용한 일자 별 가계 내역 출력

CRUD

- JSTL core의 태그 사용, 변수명에 값을 할당하여 View에 출력

- Listener를 활용하여 현재 / 일일 / 누적 접속자 수를 확인 가능
- Session 소멸 시 console에 표시, 현재 접속자 수 감소 처리
